Virtualisation

Objectifs

Créer une infrastructure virtualisée pour tester des environnements multiples, simuler un réseau et améliorer l'efficacité des ressources. Ce projet est utile pour les entreprises cherchant à réduire les coûts matériels tout en augmentant leur flexibilité dans la gestion des systèmes.

Technologies utilisées

  • VMware Workstation pour la gestion des machines virtuelles (VM).
  • Réseaux virtuels pour connecter les machines entre elles.
  • Utilisation de Windows Server et diverses distributions Linux (Ubuntu, CentOS).

Les étapes principales

  1. Installation et configuration de VMware Workstation.
  2. Création de plusieurs machines virtuelles avec des OS différents.
  3. Configuration d'un réseau interne virtuel pour tester les scénarios réseau.
  4. Mise en œuvre d'un projet, notamment le test de scénarios de pare-feu, DNS et AD.
  5. Validation des résultats et rédaction d'un rapport d'évaluation.
